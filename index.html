Marketing Analytics Machine Learning Tests A/B

OptiBank – Dashboard de pilotage marketing assisté par l'IA

Contexte : Analyser plus de 41 000 enregistrements d'une banque pour optimiser ses campagnes marketing.

Outils : Python (Pandas, scikit-learn), Plotly Dash, SQLite.

Solution : Développement d'un modèle prédictif (Random Forest, Régression Logistique) avec une précision de 89% pour anticiper les souscriptions, couplé à un dashboard temps réel. Validation statistique via tests A/B (p-value < 0.001).

L'Impact (So What ?)

Outil de pilotage qui permet à l'équipe marketing de passer de l'intuition à une décision stratégique basée sur la data. Réduction du temps d'analyse de plusieurs heures hebdomadaires, et recommandations concrètes pour éviter le gaspillage budgétaire sur les profils à faible conversion.

Akɔwɛdagbe – Plateforme d'analyse des dépenses ménagères

Contexte : Améliorer la gestion budgétaire des ménages au Bénin face aux cycles de revenus irréguliers.

Outils : Python, Flask, SQLite, Chart.js, HTML/CSS.

Solution : Application web MVP full-stack avec un modèle de Régression Linéaire embarqué pour prédire les dépenses futures à partir des habitudes passées, couplée à une interface visuelle (Chart.js).

L'Impact (So What ?)

Contribution directe à l'inclusion financière : les utilisateurs passent d'une gestion informelle à un pilotage structuré de leur budget. Capacité d'anticiper les dépenses sur 1 à 3 mois, réduisant ainsi la précarité financière à court terme.

Bellabeat (2024) – Analyse comportementale utilisateurs

Contexte : Dégager des insights comportementaux à partir de données réelles Fitbit pour orienter la stratégie produit d'objets connectés féminins.

Outils : R, Tidyverse, ggplot2.

Solution : Analyse exploratoire complète (EDA) des données Fitbit. Identification d'une forte asymétrie d'activité (81% concentrée sur 3 jours) et d'une corrélation exploitable entre sommeil et engagement.

L'Impact (So What ?)

Traduction de données complexes en 4 recommandations marketing actionnables permettant d'optimiser le ciblage des notifications push et d'améliorer concrètement l'engagement autour du produit Bellabeat Time.
